# briefly-ai
Serverless Voice-to-Action tool utilizing AWS Bedrock (Claude 3) &amp; Transcribe to convert audio into structured executive summaries.


Nombre del Proyecto: Briefly AI - Serverless Voice-to-Action Tool

Descripción: Aplicación Fullstack Serverless que utiliza AWS Transcribe y Amazon Bedrock (Claude 3) para transformar notas de voz desestructuradas en resúmenes ejecutivos y listas de tareas (Action Items) automáticamente.

Stack Tecnológico:

Frontend: Vanilla JS, CSS3 (Glassmorphism UI).

Backend: AWS Lambda (Python), DynamoDB.

AI/ML: AWS Bedrock (Generative AI), Amazon Transcribe.

Storage: Amazon S3.

Caso de Uso: Optimización de flujos de trabajo para Project Managers. Elimina la necesidad de tomar notas manuales durante las reuniones.
